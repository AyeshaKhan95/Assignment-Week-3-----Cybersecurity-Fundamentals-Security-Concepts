# Assignment-Week-3---Cybersecurity-Fundamentals-Security-Concepts
This document is to understand the computer networking basic cybersecurity concepts through both theoretical research and hands-on practice using Kali Linux and explore common network protocols, analyze secure web communication, basic network reconnaissance, investigate DNS records, and information about websites using industry-standard tools.
This will develop practical skills in network analysis, command-line usage, and information gathering while improving ability to identify common security risks and understand how modern networks and web applications communicate.

This document will provide understanding the followings:  
The purpose and usage of common network protocols and their default port numbers.
Differentiating between HTTP and HTTPS and the importance of secure web communication.
Using basic Kali Linux networking and reconnaissance tools such as ping, nmap, nslookup, dig, whatweb, and whois.
Performing basic network connectivity and service discovery on authorized targets.
Understanding the DNS resolution process and how domain information is retrieved.
Research and explaining the complete process that occurs when a user accesses a website through a web browser.
Providing a networking cheat sheet for quick reference to protocols, ports, and common security considerations.
Developing practical troubleshooting, analytical, and reporting skills using real-world cybersecurity tools.
